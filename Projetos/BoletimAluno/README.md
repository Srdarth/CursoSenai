# 📚 Boletim do Aluno

## Sobre

Projeto desenvolvido durante o curso de Backend Java no SENAI.

O objetivo foi praticar entrada de dados, cálculos e estruturas condicionais.

---

## Conceitos utilizados

- Scanner
- Variáveis
- Tipos primitivos
- Operadores matemáticos
- Média aritmética
- if / else

---

## Funcionalidades

O programa permite:

- Informar o nome do aluno
- Informar as notas
- Calcular a média
- Mostrar situação do aluno

---

## O que aprendi

Neste projeto pratiquei:

- Entrada de dados
- Cálculos utilizando variáveis
- Comparações
- Tomada de decisão utilizando if

---

## Melhorias futuras

- Cadastro de vários alunos
- Utilização de arrays
- Utilização de listas
- Cadastro de disciplinas
