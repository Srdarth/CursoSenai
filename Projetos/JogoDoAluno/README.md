# 🎲 Jogo do Aluno

## Sobre

Projeto desenvolvido durante o curso de Backend Java no SENAI.

Neste exercício foi criado um jogo de adivinhação onde o computador sorteia um número aleatório e o usuário tenta acertá-lo.

Foi o primeiro contato com geração de números aleatórios utilizando a classe `Random`.

---

## Conceitos utilizados

- Scanner
- Random
- Variáveis
- If / Else
- While
- Operadores relacionais
- Entrada de dados

---

## Funcionalidades

O programa permite:

- Sortear um número aleatório.
- Receber o número escolhido pelo usuário.
- Verificar se o usuário acertou.
- Informar o número correto quando houver erro.
- Perguntar se o usuário deseja jogar novamente.

---

## O que aprendi

Neste projeto aprendi:

- Como utilizar a classe `Random`.
- Como criar jogos simples.
- Como controlar repetições utilizando `while`.
- Como validar respostas do usuário.
