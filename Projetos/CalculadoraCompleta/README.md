# 🧮 Calculadora Completa

## Sobre

Este foi um dos primeiros projetos desenvolvidos durante o curso de Backend Java no SENAI.

O objetivo foi praticar os conceitos básicos da linguagem Java utilizando interação com o usuário através do Scanner.

---

## Conceitos utilizados

- Variáveis
- Scanner
- Operadores matemáticos
- Estruturas condicionais (if/else)
- Estrutura de repetição (while)
- Menu interativo
- Controle utilizando variável booleana

---

## Funcionalidades

A calculadora permite ao usuário:

- Somar
- Subtrair
- Multiplicar
- Dividir
- Continuar utilizando o sistema até decidir sair

---

## O que aprendi

Durante este projeto aprendi:

- Como receber dados do usuário
- Como criar menus
- Como utilizar loops
- Como controlar o encerramento do programa
- Como organizar um programa simples

---

## Melhorias futuras

- Utilizar métodos
- Utilizar classes
- Histórico de operações
- Tratamento de erros
