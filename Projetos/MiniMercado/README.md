# 🏪 Mini Mercado

## Sobre

Projeto desenvolvido durante o curso de Backend Java no SENAI.

Este projeto reúne praticamente todos os conceitos estudados até o momento, funcionando como um pequeno sistema de gerenciamento de produtos.

---

## Conceitos utilizados

- Scanner
- Arrays
- Arrays paralelos
- While
- For
- Switch
- If / Else
- Operadores matemáticos

---

## Funcionalidades

O programa permite:

- Listar produtos
- Buscar produtos por nome ou índice
- Mostrar produtos pares
- Mostrar produtos ímpares
- Calcular preço médio
- Encontrar produto mais caro
- Encontrar produto mais barato
- Aplicar desconto em todos os produtos

---

## O que aprendi

Durante este projeto consegui reunir praticamente todos os conteúdos vistos até o momento do curso.

Também aprendi a dividir melhor a lógica em pequenos blocos e a resolver problemas utilizando programação.
