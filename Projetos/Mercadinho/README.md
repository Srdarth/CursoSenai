# 🛒 Mercadinho

## Sobre

Projeto desenvolvido durante o curso de Backend Java no SENAI.

Este foi um dos primeiros sistemas utilizando arrays paralelos para representar um pequeno mercado, permitindo consultar produtos e seus respectivos preços.

---

## Conceitos utilizados

- Scanner
- Arrays
- Arrays paralelos
- Estruturas condicionais
- Estruturas de repetição
- Operadores

---

## Funcionalidades

O programa permite:

- Listar produtos
- Exibir preços
- Consultar um produto específico
- Navegar pelas opções do sistema

---

## O que aprendi

Neste projeto aprendi:

- Como relacionar informações utilizando arrays paralelos.
- Como percorrer arrays com estruturas de repetição.
- Como criar menus simples.
- Como organizar melhor a lógica do programa.

---

## Melhorias futuras

- Cadastro de novos produtos
- Alteração de preços
- Exclusão de produtos
- Utilização de métodos
- Utilização de classes
