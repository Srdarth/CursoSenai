# 🎯 Sorteio do Aluno

## Sobre

Projeto desenvolvido durante o curso de Backend Java no SENAI.

O objetivo deste exercício foi praticar a geração de números aleatórios e reforçar o uso das estruturas condicionais e de repetição.

---

## Conceitos utilizados

- Scanner
- Random
- While
- If / Else
- Operadores

---

## Funcionalidades

O programa permite:

- Gerar números aleatórios.
- Comparar o número escolhido pelo usuário.
- Exibir mensagens de acerto ou erro.
- Repetir o jogo até o usuário decidir encerrar.

---

## O que aprendi

Durante este projeto pratiquei:

- Uso da classe `Random`.
- Estruturas de repetição.
- Comparações.
- Organização da lógica do programa.
