# 🛍️ Mercadinho Avançado

## Sobre

Projeto desenvolvido durante o curso de Backend Java no SENAI.

Nesta versão o sistema ficou mais completo, reunindo diversos conceitos estudados até o momento em um único programa.

---

## Conceitos utilizados

- Scanner
- Arrays
- Arrays paralelos
- While
- For
- If / Else
- Switch
- Operadores

---

## Funcionalidades

O programa permite:

- Listar produtos
- Buscar produtos
- Exibir produtos pares e ímpares
- Consultar preços
- Navegar por um menu interativo

---

## O que aprendi

Neste projeto pratiquei:

- Organização de menus.
- Estruturas de decisão.
- Estruturas de repetição.
- Manipulação de arrays.
- Desenvolvimento de programas maiores.
