# 🛍️ Gestor de Compras

## Sobre

Projeto desenvolvido durante o curso de Backend Java no SENAI.

Este exercício apresentou a diferença entre Arrays e ArrayList, mostrando quando cada estrutura pode ser utilizada.

Foi o primeiro contato com coleções em Java.

---

## Conceitos utilizados

- Arrays
- ArrayList
- List
- For
- For-each

---

## Funcionalidades

O programa permite:

- Trabalhar com Arrays.
- Trabalhar com ArrayList.
- Percorrer coleções.
- Exibir informações armazenadas.

---

## O que aprendi

Durante este projeto aprendi:

- Diferenças entre Array e ArrayList.
- Como adicionar elementos em listas.
- Como percorrer coleções.
- Quando utilizar cada estrutura.
