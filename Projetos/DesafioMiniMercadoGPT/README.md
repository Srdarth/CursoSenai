# 🏪 Desafio Mini Mercado GPT

## Sobre

Projeto desenvolvido como desafio de revisão durante o curso de Backend Java no SENAI.

O objetivo foi reunir praticamente todos os conceitos estudados até aquele momento em um único sistema.

Este projeto foi utilizado para consolidar o aprendizado antes da introdução de novos conteúdos.

---

## Conceitos utilizados

- Scanner
- Arrays
- Arrays paralelos
- While
- For
- If / Else
- Switch
- Operadores matemáticos
- Organização de menus

---

## Funcionalidades

O programa permite:

- Listar produtos.
- Buscar produtos.
- Exibir preços.
- Navegar por menus.
- Trabalhar com estruturas de repetição.
- Manipular arrays.

---

## O que aprendi

Este desafio consolidou praticamente todo o conteúdo estudado até o momento.

Foi importante para entender como diferentes conceitos trabalham juntos em um único programa.
